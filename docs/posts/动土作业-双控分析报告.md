## 执行摘要

动土作业是市政与建筑工程中涉及地表开挖、地下管线探挖、基坑/沟槽开挖与支护的基础性施工环节，涵盖机械开挖、人工开挖、基坑支护、降水作业、土方运输和回填压实等多项工序，涉及挖掘机、破碎锤、运输车辆、打夯机、水泵等多种施工设备，具有地下条件不确定性大、隐蔽设施风险高、机械交叉作业密集、地质水文条件影响显著等特点，是施工安全事故的多发环节。

本次分析采用蝴蝶结模型对动土作业进行双重预防机制结构化分析，通过三维扫描法识别出**10个风险点**、**6个危险源**、**7个初始事件**、**21个威胁**、**7种事故类型**、**13个后果**，设计**21道控制措施**、**16道应急措施**和**37项关键保障活动**，形成覆盖动土作业全场景的闭环管控体系。

经LS评估，动土作业共识别出**重大风险5项**（沟槽坍塌人员被埋、挖掘机倾覆司机伤亡、运输车辆碰撞人员、挖掘机回转碰撞人员、燃气管道破坏爆炸火灾）、**较大风险6项**（高压电缆破损触电伤亡、基坑涌水工期延误、人员触电伤亡、地下管线破坏通信中断、人员中毒窒息、挖掘机碰撞沟内人员）、**一般风险2项**（设备损坏、管线路由中断间接损失），其中沟槽坍塌、挖掘机作业和地下管线保护为需重点管控的核心风险。

## 第一章：场景描述

### 1.1 分析背景

本次双控分析以动土作业为分析对象，采用蝴蝶结模型进行双重预防机制结构化分析。分析旨在系统识别动土作业中的危险源、威胁、初始事件、事故类型和后果，设计针对性的控制措施和应急措施，建立闭环验证机制，为动土作业的风险管控提供结构化依据。

### 1.2 工程场景

动土作业是施工总平面布置中的基础性工程，通常在项目开工初期或管线施工阶段进行，主要包括机械开挖、人工开挖/探挖、基坑/沟槽支护、降水疏干、土方运输和回填压实等核心作业。

| 特征项 | 说明 |
|--------|------|
| 作业类型 | 机械开挖、人工开挖/探挖、基坑支护、降水作业、土方运输、回填压实 |
| 主要设备 | 挖掘机、破碎锤、土方运输车辆、打夯机、水泵、发电机、振动器 |
| 关键作业 | 沟槽/基坑开挖、地下管线探挖与保护、边坡支护、降水疏干、土方外运、回填分层压实 |
| 环境条件 | 露天作业、受地质水文条件影响大、存在地下管线/电缆/燃气管道等隐蔽设施 |
| 人员配置 | 挖掘机司机、信号指挥人员、普工（人工开挖/探挖）、测量员、安全员 |
| 周边环境 | 可能涉及既有道路、建筑物、地下管线密集区、临边作业区域 |

**典型作业场景描述**：

动土作业通常在项目开工阶段或管线施工阶段进行，施工流程为：地下管线调查→测量放线→探坑开挖→机械开挖（分层/分段）→基坑/沟槽支护→降水作业→基底处理→土方运输→回填压实。基坑/沟槽开挖是动土作业的核心风险环节，开挖深度可从浅层（<2m）到深层（>5m深基坑），开挖过程中需同步进行地下管线保护、边坡支护和降水疏干。挖掘机在基坑/沟槽边缘进行挖土、装土作业，运输车辆在场地内往返运输土方，人工开挖主要用于管线探挖和基底修整。基坑/沟槽临边设置防护栏杆和警示标志，堆土区和车辆通行区与基坑边保持安全距离。整个施工过程中，地下管线（燃气、电力、通信、给排水）的探测和保护是动土作业区别于一般土方工程的特殊风险点，一旦破坏将导致次生灾害（爆炸、触电、通信中断等）。

## 第二章：风险建模

### 2.1 五要素识别

#### 2.1.1 危险源识别

采用能量源扫描法（四问法：什么有质量？什么有势能/动能？什么有压力？什么有化学能？）识别危险源。

```
危险源识别结果
  │
  ├── H-01 基坑/沟槽边坡土体 [势能]
  │     [分析逻辑：开挖后的边坡土体处于应力平衡临界状态，具有坍塌势能，一旦土体应力平衡破坏将导致边坡失稳坍塌]
  │
  ├── H-02 挖掘机/破碎锤 [动能]
  │     [分析逻辑：挖掘机作为大型移动机械设备，在作业和移动过程中具有较大动能，碰撞、倾覆或挖破管线时造成伤害]
  │
  ├── H-03 土方运输车辆 [动能]
  │     [分析逻辑：运输车辆在场地内频繁行驶，具有较大动能，车辆碰撞或碾压造成人员伤亡]
  │
  ├── H-04 地下管线/设施 [电能/化学能/压力能]
  │     [分析逻辑：地下燃气管道（化学能-易燃易爆）、高压电缆（电能）、给排水管（压力能），一旦破坏引发次生灾害]
  │
  ├── H-05 地下水/地表水 [压力能/势能]
  │     [分析逻辑：地下水位高于基底时具有水压力，降水不当或涌水可能导致边坡失稳或基底浸泡]
  │
  └── H-06 临时用电设备 [电能]
        [分析逻辑：水泵、振动器等用电设备在潮湿环境下使用，绝缘破损或漏电保护失效时造成触电伤害]
```

### 5.3 综合结论与建议

**主要结论**：

1. **沟槽边坡坍塌是动土作业的最核心风险**（R=15，重大风险），直接威胁沟槽/基坑内作业人员的生命安全。控制边坡支护质量、规范堆载管理、落实截排水措施、控制机械振动和按方案放坡是防止坍塌的六大关键防线。

2. **机械作业事故频次高、后果严重**，挖掘机倾覆（R=15）、运输车辆碰撞碾压人员（R=15）、挖掘机回转碰撞（R=15）均为重大风险。机械作业的安全管理重点在于划定安全作业区域、保障设备完好性、建立有效的信号指挥体系。

3. **地下管线保护是动土作业的特色风险**，燃气管道破坏（R=10）、高压电缆破损（R=10）、给排水管破坏涌水（R=9）均为较大风险，且次生灾害后果严重（爆炸火灾可致群死群伤），须建立管线调查→现场标识→人工探挖→机械禁挖的完整保护链条。

4. **潮湿环境下的用电安全**（R=10）需重点关注，沟槽/基坑内湿度大、积水多，用电设备绝缘下降风险高，漏电保护器的有效性和电缆保护是关键。

**改进建议**：

1. **管控层级提升建议**：当前控制措施中管理控制占比偏高（57%），建议在以下场景增加工程控制措施：
   - 深度>2m的沟槽强制采用钢板桩等支护措施
   - 大型挖掘机全面安装360°环视影像系统和倒车雷达
   - 运输车辆强制安装倒车雷达+声光报警一体装置
   - 燃气管道附近施工区域强制设置可燃气体连续监测报警装置

2. **地下管线保护体系完善建议**：
   - 建立"一探二标识三人工开挖"的标准化流程
   - 与管线产权单位建立联动响应机制
   - 配备管线探测仪、可燃气体检测仪等专业设备
   - 每段开挖前进行管线保护专项交底

3. **保障活动维度完善建议**：当前独立审计类保障活动仅占6%，建议：
   - 每月由项目安全部门对动土作业管控措施执行情况进行专项检查
   - 引入第三方监测单位对深基坑边坡变形数据进行独立复核

4. **动态更新机制建议**：
   - 动土作业环境变化大（天气、地下水位变化频繁），建议每次强降雨后、每层开挖前对风险点进行动态复核
   - 发现新的地下管线或管线位置偏差时，及时更新蝴蝶结模型和管控措施

5. **培训与能力提升**：
   - 挖掘机、破碎锤等机械操作人员必须持证上岗
   - 每季度组织一次地下管线保护专项安全培训
   - 全员掌握燃气泄漏、电缆破损等紧急情况的应急处置流程
