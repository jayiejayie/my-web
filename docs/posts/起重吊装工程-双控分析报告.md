# 起重吊装工程 蝴蝶结双控分析报告

## 执行摘要

起重吊装工程是建筑施工中危险性最高的专项工程之一，涉及塔式起重机、汽车起重机、履带起重机等多种设备，以及钢构件、预制构件、管道、设备等多种吊装对象。GB 6441-2025将"起重致害"列为独立事故类型，据统计建筑行业每年起重吊装事故造成大量人员伤亡和财产损失。

本次分析围绕起重吊装工程，采用蝴蝶结模型进行双重预防机制结构化分析。识别出**6个风险点、7个危险源、10种初始事件、20个威胁、6种事故类型、10种后果**，设计了**20道控制措施、11道应急措施、31项关键保障活动**。通过纵深防御为起重吊装工程提供全面的风险管控方案。

## 第一章：场景描述

### 1.1 分析背景



起重吊装工程贯穿建筑施工全过程，从基础阶段的钢筋笼、预制桩吊装，到主体阶段的钢结构、预制构件吊装，再到装修机电阶段的设备、管道吊装，涉及面广、频次高。起重吊装事故具有突发性强、后果严重、群死群伤风险高的特点。本报告以建筑施工起重吊装工程为分析对象，按照双重预防机制要求，围绕"起重致害、物体打击、高处坠落、坍塌、触电、机械致害"六类事故类型进行系统分析，为现场起重吊装作业的风险管控提供指导。

### 1.2 工程场景

起重吊装工程典型作业场景涵盖以下特征：

| 要素 | 内容 |
|------|------|
| 作业类型 | 塔吊起重吊装、汽车吊吊装、大型构件吊装、设备吊装就位 |
| 关键设备 | 塔式起重机、汽车起重机、履带起重机、门式起重机、桅杆式起重机 |
| 吊装对象 | 钢柱、钢梁、预制构件、预制桩、管道、设备、钢筋笼 |
| 吊装方式 | 单机吊装、双机抬吊、多机抬吊、分件吊装、整体吊装 |
| 人员配置 | 起重司机、信号司索工、起重指挥、安全监护人员 |
| 环境条件 | 露天作业，受风力、雨雪天气影响，常与多工种交叉作业 |
| 周边环境 | 吊臂回转范围内存在既有建筑物、架空线路、临边通道等 |

**典型作业场景描述**：在建筑施工现现场，塔式起重机或汽车起重机进行构件吊装作业。起重司机操作起重机，信号司索工通过旗语/对讲机进行指挥，起吊混凝土预制构件或钢构件从地面起吊至安装位置。吊装过程中，吊物经过临时通道、堆料区、作业人员活动区域等上空，吊臂回转覆盖范围内存在交叉作业人员。设备基础或支腿地基承受巨大荷载，钢丝绳受力拉伸，各安全限位装置（力矩限制器、高度限位器、幅度限位器）处于实时工作状态。吊装作业受风速、能见度等自然条件影响，同时与周边焊接、安装等施工活动存在空间冲突。

|:----:|:----------:|--------|:----------:|---------|

## 第二章：风险建模

### 2.1 五要素识别

**分析逻辑**：危险源→初始事件→威胁→事故类型→后果，形成完整闭环推演链。先识别能量载体（危险源），再判断其失控方式（初始事件），从人-物-环维度找出触发因素（威胁），根据能量释放方式确定事故类型，最后推断最终损失（后果）。

#### 2.1.1 危险源识别

采用能量源扫描法识别危险源。起重吊装工程以"重力势能"为主要能量类型，同时涉及电能、动能等：

| 编号 | 危险源 | 关联风险点 | 能量类型 | 分析逻辑 |
|:----:|--------|:----------:|:--------:|---------|
| H-01 | 起重吊物（被吊构件/材料） | RP-001/002/003 | 势能 | 吊离地面的构件具有重力势能，一旦失控从高处坠落，势能转化为动能可造成致命打击伤害 |
| H-02 | 塔式起重机钢结构（塔身/臂架/平衡臂） | RP-004 | 势能 | 塔身高耸、臂架和平衡臂自重及配重产生巨大倾覆力矩，若基础失效或钢结构失稳将导致整体倒塌 |
| H-03 | 汽车起重机臂架及配重 | RP-005 | 势能 | 臂架和配重产生的倾覆力矩超过支腿支撑范围，支腿地基失效或超载将导致整车倾覆 |
| H-04 | 起重钢丝绳及吊具 | RP-001/002/003 | 拉力能 | 钢丝绳和吊具承载全部起重量，一旦断裂释放弹性势能，吊物瞬间坠落 |
| H-05 | 起重机基础/轨道地基 | RP-004 | 承载势能 | 塔吊基础或汽车吊支腿地基承受巨大荷载，地基失稳导致起重机倾斜倒塌 |
| H-06 | 高压架空输电线路 | RP-001/002 | 电能 | 吊臂或钢丝绳靠近高压线时，空气间隙击穿放电，产生电弧触电及短路火灾 |
| H-07 | 起重机回转机构/卷扬机构 | RP-004/005 | 动能 | 回转平台旋转带动平衡臂和起重臂高速转动，回转范围内人员被挤压碰撞；卷扬机构卷入衣物肢体 |

#### 2.1.2 初始事件识别

初始事件是危险源从"受控"到"失控"的临界状态：

| 编号 | 初始事件描述 | 关联危险源 | 失控状态描述 | 分析逻辑 |
|:----:|-------------|:----------:|------------|---------|
| IE-01 | 吊物从高处坠落 | H-01 起重吊物 | 吊物脱离控制自由下落 | 吊物因超载、钢丝绳断裂、脱钩等原因失去约束，重力和高度使其坠落冲击 |
| IE-02 | 塔式起重机发生倾覆/倒塌 | H-02 塔吊钢结构 | 塔身及臂架整体失去稳定 | 基础失效、附着破坏或超载力矩使塔吊整体失稳，表现为整机倒塌 |
| IE-03 | 汽车起重机发生倾覆 | H-03 汽车吊臂架及配重 | 整车绕支腿倾翻 | 支腿地基沉降或超载力矩超过稳定力矩，整车向一侧倾翻 |
| IE-04 | 起重钢丝绳发生断裂 | H-04 钢丝绳及吊具 | 钢丝绳承载能力丧失 | 钢丝绳因磨损、腐蚀、过载等因素达到破断极限，突然断开 |
| IE-05 | 起重臂发生折断 | H-02/H-03 臂架结构 | 臂架钢结构失效变形 | 臂架因疲劳、超载或碰撞导致应力超限，结构破坏折断 |
| IE-06 | 吊臂或钢丝绳触碰高压线放电 | H-06 高压架空线路 | 安全距离不足空气击穿 | 吊臂或钢丝绳进入高压线安全距离不足区域，产生电弧放电 |
| IE-07 | 塔吊基础连接失效 | H-05 起重机基础/地基 | 塔身与基础连接部位失效 | 地脚螺栓断裂、基础混凝土开裂或地基不均匀沉降导致塔身与基础连接失效、塔身歪斜 |
| IE-08 | 人员从高处坠落 | H-02 塔吊钢结构 | 人员在攀爬或检修时失稳 | 塔吊操作人员上下塔吊、检修平台作业时未使用防坠器滑落 |
| IE-09 | 回转机构挤压人员 | H-07 回转机构 | 回转运动挤压入侵人员 | 回转机构旋转时人员误入回转半径内被平衡臂或配重挤压 |
| IE-10 | 卷扬机构绞伤人员 | H-07 卷扬机构 | 人员肢体被卷入卷扬 | 卷扬钢丝绳排列作业中人员手套或衣物被卷入卷筒 |

#### 2.1.5 后果识别

从"人员伤亡、财产损失、环境影响、社会影响"维度评估：

| 编号 | 后果描述 | 关联危险源 | 关联初始事件 | 关联事故类型 | 分析逻辑 |

### 5.3 综合结论与建议

**主要结论**：

1. **起重吊装工程风险特征**：存在7个危险源（4种能量类型）、20个威胁（人8/物9/环3）、10种初始事件、6种事故类型，风险呈"高后果-中高可能"特征，属于建筑施工最危险的专项工程之一。

2. **核心高风险场景2个**：
   - 吊物坠落（R=15，高风险）——吊物频繁经过作业人员上方，力矩限制器+制动器+吊具完整性是核心屏障
   - 汽车吊倾覆（R=15，高风险）——支腿地基沉降和超载是主要威胁，路基箱+力矩限制器是核心屏障

3. **事故类型以"起重致害"居首**，但经事故类型反查补充了高处坠落、触电、机械致害三种事故类型，这些类型在常规分析中易被忽略。

4. **屏障薄弱环节识别**：
   - 塔吊倒塌后果极严重（R=10），但目前地脚螺栓松动和附着失效的检查依赖人工月度检查，建议增加在线监测
   - 高压线放电虽L=2但后果致命，建议高风险区域增设空间限位雷达装置
   - 高处坠落防护（防坠器+安全带）为PPE层级，建议在塔吊爬梯增设安全绳导轨

**管理建议**：

| 优先级 | 建议 | 类型 | 说明 |
|:-----:|------|:----:|------|
| 立即 | 力矩限制器、高度限位器每日开工前试运行验证 | 强制 | 所有起重机械必须安装合格的安全限位装置，每日验证并记录 |
| 立即 | 钢丝绳日检制度+报废台账 | 强制 | 每班次检查钢丝绳状态，达到报废标准立即更换并记录 |
| 短期 | 塔吊基础地脚螺栓增加在线监测传感器 | 改善 | 实时监测螺栓预紧力变化，预防松动导致的连接失效 |
| 短期 | 高压线保护区吊装增设空间限位雷达+声光报警 | 改善 | 吊臂距高压线达安全距离临界值时自动报警并停止变幅 |
| 短期 | 塔吊爬梯全线设置垂直安全绳导轨 | 改善 | 替代"每段爬梯分别挂防坠器"的低效模式，全程连续防护 |
| 长期 | 推广"无人化"吊装：遥控操作+视频监控+AI限位 | 创新 | 减少人员暴露，降低高处坠落和物体打击风险 |
| 长期 | 建立起重机械设备全生命周期数字化管理平台 | 体系 | 覆盖设备进场、安装、使用、检测、维保、拆除全过程 |
