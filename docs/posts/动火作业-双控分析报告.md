## 执行摘要

动火作业是市政工程施工中风险等级最高的作业类型之一，涵盖管道焊接（钢管焊接/PE管热熔）、钢结构焊接与切割、钢筋焊接与切割、防水卷材热熔施工、气焊/气割等核心作业，涉及电焊机、气瓶（氧气/乙炔/丙烷）、热熔焊机等多种动火设备，电弧温度可达3000℃以上，焊渣温度超过1000℃，具有明火温度高、火花飞溅范围大、作业点分散、受限空间和高处动火叠加等特点，火灾、爆炸、触电、灼烫事故风险突出。本次分析共识别**13个风险点、7个危险源、20个威胁、9个初始事件、9种事故类型、14个后果**，设计**20道控制措施、13道应急措施、38项关键保障活动**，形成覆盖动火作业全场景的闭环管控体系。经LS评估，高风险5项（焊接引燃火灾R=16、火灾人员伤亡R=15、乙炔泄漏爆炸R=15、焊工触电R=15、高处动火坠落R=15），中风险6项，低风险3项，其中焊接火花引燃火灾为需重点管控的核心风险。

## 第一章：场景描述

### 1.1 分析背景

本次分析以市政工程中的动火作业为分析对象，基于动火作业的施工特点和典型场景，采用蝴蝶结模型进行双重预防机制结构化分析。动火作业是市政工程中最常见的危险性作业之一，根据《建设工程施工现场消防安全技术规范》（GB 50720-2011）和《焊接与切割安全》（GB 9448-1999）的要求，动火作业必须实行审批制度，但在实际执行中仍存在管控不到位的情况。分析旨在系统识别动火作业中的危险源、威胁、初始事件、事故类型和后果，设计针对性的控制措施和应急措施，建立闭环验证机制，为动火作业的风险管控提供结构化依据。

### 1.2 工程场景

| 要素   | 内容                                                                                                 |
| ---- | -------------------------------------------------------------------------------------------------- |
| 作业类型 | 钢管焊接（手工电弧焊/CO₂气体保护焊）、PE管热熔焊接、钢结构焊接与切割（气割/等离子切割）、钢筋焊接（闪光对焊/电渣压力焊/电弧焊）、防水卷材热熔施工（SBS/APP改性沥青卷材）、气焊/气割 |
| 主要设备 | 电焊机（逆变焊机/交流焊机）、氩弧焊机、CO₂气体保护焊机、热熔焊机、氧气瓶/乙炔瓶/丙烷瓶、焊炬/割炬、角磨机/切割锯、等离子切割机、热熔喷灯                           |
| 焊接材料 | 焊条（E43/E50系列）、焊丝（实心/药芯）、焊剂、氧气、乙炔/丙烷、氩气、CO₂气体                                                       |
| 作业环境 | 露天作业为主，部分在沟槽底部、检查井内、高处脚手架平台、桥面、屋面、泵房内等场所，受天气影响大                                                    |
| 人员配置 | 焊工（持特种作业操作证）、气焊工、热熔焊工、监火人、安全员                                                                      |

**典型作业场景描述**：

市政工程动火作业场景多样。钢管焊接通常在沟槽底部进行，焊工在有限空间内操作电焊机，焊接电弧产生高温和飞溅焊渣，焊条药皮燃烧产生焊接烟尘（含锰、铬、镍等金属氧化物），沟底通风条件差时烟尘积聚严重。钢结构焊接与切割多在脚手架或高处平台上进行，焊渣从高处掉落可引燃下方可燃物，同时高处作业存在坠落风险。防水卷材热熔施工使用喷灯或热熔机加热卷材背面（温度160~180℃），火焰直接接触防水卷材，操作不当极易引燃卷材或周围可燃物。气焊/气割使用氧气和乙炔（或丙烷），氧气瓶满瓶压力约15MPa，乙炔瓶满瓶压力约2.5MPa，均为压力容器，乙炔为易燃易爆气体，回火、泄漏、混触均可引发爆炸。PE管热熔焊接使用热熔焊机加热管材端面至熔融状态（温度200~230℃），高温加热板直接接触熔融塑料，操作不当可导致塑料过度分解产生有毒烟气。

  │     │     [分析逻辑：焊炬/割炬回火火焰沿乙炔胶管倒灌进入乙炔瓶内，乙炔在瓶内分解爆炸]
  │     │
  │     └── IE-05 氧气瓶阀或附件沾染油脂发生燃烧
  │           [分析逻辑：氧气瓶阀、减压器或胶管沾染油脂，在高压纯氧环境中自燃引发火灾/爆炸]
  │
  ├── H-04 切割砂轮片/锯片
  │     └── IE-06 砂轮片在切割中碎裂飞出
  │           [分析逻辑：砂轮片受冲击或安装不当导致碎裂，高速旋转碎片由"约束状态"转为"无约束飞散"]
  │
  ├── H-05 热熔设备/喷灯高温部件
  │     └── IE-07 高温部件直接接触作业人员皮肤
  │           [分析逻辑：热熔焊机加热板或喷灯火焰意外接触作业人员身体，高温热量传导至皮肤组织致伤]
  │
  ├── H-06 焊接烟尘/塑料分解烟气
  │     └── IE-08 焊接烟尘/有毒烟气在受限空间积聚超限
  │           [分析逻辑：受限空间内通风不足，焊接烟尘或塑料分解烟气持续产生无法排出，浓度随时间累积超限]
  │
  └── H-07 高处动火作业位置
        └── IE-09 作业人员从高处失稳坠落

#### 2.1.3 威胁识别

### 5.3 综合结论与建议

**主要结论**：

1. **火灾是动火作业的最核心风险**（C-01，R=16，高风险），施工现场大量存在的木模板、保温材料、防水卷材、土工布等可燃物为火灾提供了物质基础。焊接火花引燃、高处焊渣掉落引燃下方堆场（C-13，R=12）和焊接结束阴燃复燃是三种主要火灾触发模式。动火作业前可燃物清理、接火措施和看火人制度是防止火灾的三大关键防线。

2. **气瓶系统爆炸危害最大**（C-03，R=15，高风险），乙炔/丙烷泄漏后与空气形成爆炸性混合物，一旦遇明火将发生化学爆炸，可能造成群死群伤。气瓶胶管老化泄漏、气瓶距动火点过近和回火倒灌是三种主要触发路径。回火防止器、气瓶安全距离管理和每日检漏制度是防止爆炸的核心屏障。

3. **焊工触电在潮湿环境焊接中风险突出**（C-06，R=15，高风险），沟槽底部焊接作业环境潮湿，电缆在沟底拖拽易破损，漏电保护器失效和绝缘装备缺失时漏电风险极高。焊工触电后在高处可能发生坠落二次伤害（C-07，R=10）。漏电保护器每日测试和绝缘装备完好是保障焊工生命安全的底线。

4. **高处动火作业双重风险叠加**（C-12，R=15，高风险），高处动火同时面临高处坠落和火灾两大风险，且动火操作本身增加了作业人员失稳的可能性。安全带高挂低用、安全平网和接火盆是关键技术屏障。

5. **受限空间动火作业中毒窒息风险不可忽视**（C-08，R=12中风险；C-09，R=10中风险），检查井、管沟等受限空间内焊接作业产生的焊接烟尘和CO在通风不良时快速积聚，可在短时间内达到致害浓度。强制通风是唯一有效的工程控制手段。
