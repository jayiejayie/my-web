# 风险管控清单与隐患排查清单编制指南（V2 精炼版）

> **一个案例讲透双重预防机制的两张清单**
> 
> 以某市政道路排水工程管沟开挖为线索，串联全部概念、步骤和要点

---

## 写在前面：从一个真实的沟槽说起

想象你正在一条城市道路下方开挖一条管沟，深度 3 米，长 800 米。地下埋着燃气管道、电力电缆、给水管线，两侧有行人车辆穿行，头顶还可能下雨。

你的任务，是**保证这条管沟从开挖到回填的全过程不出事故**。

怎么做到？这需要回答三个问题：

1. **风险在哪？**——哪些作业、哪些设备可能出事？
2. **怎么管？**——针对每个风险，该做什么来预防？
3. **管住了没有？**——措施落实了吗？有没有隐患？

"双重预防机制"就是围绕这三个问题建立的管理体系。而**风险管控清单**和**隐患排查清单**，是这个体系的两份核心文件——前者回答"风险在哪、怎么管"，后者回答"管住了没有"。

---

## 第1章 一张图看懂双重预防机制

### 1.1 一个核心逻辑

整个双重预防机制的逻辑，可以浓缩成一句话：

> **辨识风险 → 制定措施 → 编制管控清单 → 按清单排查隐患 → 治理闭环**

把这句话展开，就是十条工作步骤：

```
成立组织 → 培训教育 → 确定风险点 → 辨识危险源 → 评价风险等级
    → 制定管控措施 → 编制风险管控清单
    → 编制隐患排查清单 → 实施隐患排查治理 → 持续改进
```

### 1.2 两份清单的关系

这是理解整个体系的关键：

| | 风险管控清单 | 隐患排查清单 |
|---|---|---|
| **解决的问题** | 风险在哪？怎么管？ | 措施落实了吗？ |
| **核心内容** | 风险点→危险源→等级→措施 | 排查什么？怎么查？多久查？ |
| **逻辑关系** | 是源头和基础 | 是验证和落地 |
| **一句话** | 告诉我该做什么 | 检查我做了没有 |

**最关键的一句话：管控措施就是排查内容。**

也就是说，你在风险管控清单里写下的每一条措施，都要在隐患排查清单里对应一条检查项。这就是"双重"的含义——风险管控是"第一道防线"，隐患排查是"第二道防线"。

### 1.3 几个必须说清楚的概念

| 概念 | 通俗理解 | 示例 |
|------|---------|------|
| **风险点** | "可能出事的地方或活动" | 管沟开挖这个作业、挖掘机这台设备 |
| **危险源** | "导致出事的根源" | 沟壁松动的土块、失效的液压管路 |
| **风险等级** | "出事的可能性×后果的严重性" | Ⅰ级（红）/Ⅱ级（橙）/Ⅲ级（黄）/Ⅳ级（蓝） |
| **管控措施** | "为了不出事采取的办法" | 放坡、支护、安全交底、戴安全帽 |
| **事故隐患** | "该做的没做到" | 没放坡、没戴安全帽、支护不及时 |

## 第2章 我们的案例：XX路排水工程

为了把抽象的方法讲清楚，我们用一个具体工程贯穿全文。

| 项目 | 内容 |
|------|------|
| **工程名称** | XX市XX路排水工程 |
| **施工内容** | 雨水管道铺设（D600混凝土管，全长约800m） |
| **管沟深度** | 2.5m ~ 4.0m |
| **地质条件** | 粉质黏土，局部夹砂层，地下水位较高 |

| **施工方法** | 机械开挖为主，局部人工清底 |

这是一个非常典型的市政工程——看起来简单，但风险点不少。接下来，我们就用这个案例，一步步走完两份清单编制的全过程。

## 第3章 第一步：确定风险点（划范围）

### 3.1 什么是风险点？

风险点就是**"可能出事的地方或活动"**，分两类：

- **作业活动类（动态）**：施工中的各项作业，比如开挖、吊装、焊接
- **设备设施类（静态）**：施工中的设备、设施、场所，比如挖掘机、脚手架、沟槽区域

### 3.2 怎么找风险点？

用**三维扫描法**，从三个维度排查，确保不遗漏：

```
第一维：作业活动维度 → 按施工工序逐一扫描常规和非常规作业
第二维：设备设施维度 → 按设备台账逐一扫描各类设备设施
第三维：部位区域维度 → 按施工平面逐一扫描场所区域
```

### 3.3 案例实操

**准备工作**：收集施工图、地质勘察报告、施工组织设计、既有管线图。组建由技术负责人牵头，施工员、安全员、设备管理员参加的工作小组。

**初步划分**：按作业活动和设备设施两类划分，得到 11 个风险点。

**三维扫描排查**：在初步划分基础上，从三个维度逐一扫描：

```
管沟开挖风险点三维扫描结果（共14个风险点）
  │
  ├── 作业活动扫描（6个）
  │     RP-01 管沟机械开挖       ← 本篇主角
  │     RP-02 边坡支护安拆
  │     RP-03 管道吊装下管
  │     RP-04 管道焊接与接口处理
  │     RP-05 管沟回填及压实作业
  │     RP-06 沟槽上下通行及物料运输
  │
  ├── 设备设施扫描（5个）
  │     RP-07 挖掘机/装载机
  │     RP-08 排水设备（水泵/发电机）
  │     RP-09 边坡支护结构
  │     RP-10 沟槽临边防护及上下通道设施
  │     RP-11 打夯机/压路机
  │
  └── 部位区域扫描（3个，补充发现）
        RP-12 地下管线交叉区域     ← 容易被遗漏！
        RP-13 汛期/雨天影响区域     ← 容易被遗漏！
        RP-14 管沟周边堆土超载区域   ← 容易被遗漏！
```

**关键发现**：通过部位区域扫描，补充了 3 个容易被遗漏的风险点。这说明**只看作业和设备是不够的，还要看"场所和环境"**。

### 3.4 要点提示

| 要点 | 说明 |
|------|------|
| **大小适中** | 风险点粒度不宜过大或过小——过大则辨识困难，过小则清单臃肿 |
| **分类清晰** | 作业活动和设备设施两类要明确区分，避免交叉重叠 |
| **覆盖全面** | 必须覆盖所有施工阶段、所有区域、所有设备 |
| **动态更新** | 随施工进展，新阶段可能带来新风险点 |

## 第4章 第二步：辨识危险源（找根源）

### 4.1 什么是危险源辨识？

风险点只是一个"位置"或"活动"。在这个位置上，具体**什么东西可能导致事故**？这就是危险源。

危险源分四类：人的因素、物的因素、环境因素、管理因素。

辨识方法也是两套：

| 风险点类型 | 辨识方法 | 原理 |
|-----------|---------|------|
| 作业活动类 | **JHA法**（作业危害分析） | 把作业拆成步骤，分析每一步的危险 |
| 设备设施类 | **SCL法**（安全检查表法） | 列出检查项目，分析每项的偏差后果 |

### 4.2 JHA法案例：管沟机械开挖（RP-01）

把"管沟机械开挖"拆成三个步骤，逐一分析：

```
步骤1：施工准备与测量复核
  ├── [人] 作业人员未经安全交底、无证上岗
  ├── [物] 测量仪器未校准、管线探测仪失灵
  ├── [环] 管线图纸与实际位置不符
  ├── [管] 管线保护交底不清、未办理动土审批
  └── 潜在事故：挖断燃气管道（爆炸）、挖破电力电缆（触电）

步骤2：机械分层开挖
  ├── [人] 挖掘机司机疲劳作业、违章操作
  ├── [物] 液压系统故障、挖掘机距沟边过近
  ├── [物] 边坡未按设计放坡、土质不良未加固
  ├── [环] 降雨土体含水率增大、回转半径内有人进入
  ├── [管] 未进行边坡稳定性监测
  └── 潜在事故：坍塌掩埋、机械伤害、物体打击

步骤3：人工清底修坡
  ├── [人] 未佩戴安全帽/安全带、沟槽内站位不当
  ├── [物] 沟壁悬空土块坠落、未设专用上下通道
  ├── [环] 沟壁渗水软化、沟底有害气体积聚
  └── 潜在事故：坍塌掩埋、高处坠落、中毒窒息
```

### 4.3 SCL法案例：挖掘机（RP-07）

对设备设施，列出检查项目和标准要求，分析偏差后果：

| 检查项目 | 标准要求 | 偏差后果 |
|---------|---------|---------|
| 整机资质 | 铭牌清晰、检验合格证有效 | 机械伤害 |
| 液压系统 | 管路无渗漏、液压油位正常 | 物体打击 |
| 安全防护装置 | 驾驶室FOPS完好、倒车影像有效、喇叭/警灯正常 | **车辆伤害** |
| 工作装置 | 挖斗无开裂变形、销轴连接可靠 | 物体打击 |
| 管沟边作业适配性 | 机身距沟边≥1.5m、支腿平稳 | 机械伤害、坍塌 |

### 4.4 要点提示

| 要点 | 说明 |
|------|------|
| **人、物、环、管全覆盖** | 不要只盯着"物"，人的行为和管理制度同样重要 |
| **潜在事故类型要明确** | 用GB 6441分类，便于后续评价措施针对性 |
| **现有控制措施要记录** | 辨识时已经存在的措施要写在记录表里，为后续评价做准备 |

## 第5章 第三步：评价风险等级（定级别）

### 5.1 为什么需要评价？

辨识出了危险源，不等于所有危险源都需要同等对待。我们需要知道：**哪个风险最要命？哪个可以先放一放？**

评价方法也对应两类风险点：

| 风险点类型 | 评价方法 | 公式 |
|-----------|---------|------|
| 作业活动类 | **LEC法** | D = L（发生可能性）× E（暴露频次）× C（后果严重性） |
| 设备设施类 | **LS法** | R = L（可能性等级）× S（后果等级） |

### 5.2 LEC法案例：管沟机械开挖（RP-01）

以"挖断燃气管"这个危险源为例：

> **L（可能性）**：管线图纸不清，且未做人工探坑 → 相当可能 → 6分
> 
> **E（暴露频次）**：每天在沟槽区作业 → 每天暴露 → 6分
> 
> **C（后果）**：燃气爆炸可导致多人死亡 → 大灾难 → 40分
> 
> **D = 6 × 6 × 40 = 1440** → **Ⅰ级重大风险（红色）**

再看"人工清底时沟壁坍塌"：

> **L（可能性）**：未做敲帮问顶 → 可能，但不经常 → 3分
> 
> **E（暴露频次）**：人工在沟底作业 → 每天暴露 → 6分
> 
> **C（后果）**：掩埋可导致一人死亡 → 非常严重 → 15分
> 
> **D = 3 × 6 × 15 = 270** → **Ⅱ级较大风险（橙色）**

### 5.3 风险分级标准

| 风险等级 | D值（LEC法） | 颜色 | 管控层级 |
|---------|-------------|------|---------|
| Ⅰ级：重大风险 | D ≥ 320 | 红色 | 企业级管控 |
| Ⅱ级：较大风险 | 160 ≤ D < 320 | 橙色 | 项目部级管控 |
| Ⅲ级：一般风险 | 70 ≤ D < 160 | 黄色 | 班组级管控 |
| Ⅳ级：低风险 | D < 70 | 蓝色 | 作业人员级管控 |

### 5.4 直接判定法

以下情形可不经打分计算，**直接判定为Ⅰ级重大风险（红色）**：

1. 违反法律法规强制性条款的
2. 发生过事故且条件依然存在的
3. **超过一定规模的危大工程**
4. 构成危险化学品重大危险源的
5. 有火灾、爆炸、中毒等危险且作业人员≥10人的

> 案例中，边坡支护安拆（RP-02）和管道吊装下管（RP-03）因属于危大工程，直接判定为Ⅰ级。

### 5.5 要点提示

| 要点 | 说明 |
|------|------|
| **从严从高** | 多种方法评价结果不一致时，取最高等级 |
| **直判优先** | 符合直接判定条件的，不必再打分计算 |
| **客观公正** | L、E、C/S取值要有依据，不能随意打分 |

## 第6章 第四步：制定管控措施（定方案）

### 6.1 五大措施类型

针对辨识出的危险源，制定控制措施。措施分五类，按可靠性从高到低排列：

| 措施类型 | 通俗理解 | 示例 |
|---------|---------|------|
| **①工程技术措施** | 从硬件上消除或隔离风险 | 放坡、支护、防护栏杆、气体检测仪 |
| **②制度管理措施** | 通过制度规范行为 | 编制方案、安全交底、审批手续、日常检查 |
| **③培训教育措施** | 让人员知道风险和操作方法 | 三级教育、专项培训、班前会 |
| **④个体防护措施** | 最后一道防线 | 安全帽、安全带、绝缘鞋、防毒面具 |
| **⑤应急处置措施** | 万一出事怎么办 | 应急预案、应急物资、应急演练 |

### 6.2 措施制定的"对症下药"

不同类型的问题，用不同类型的措施解决：

```
[人] 不安全行为 → 培训教育（让他知道怎么做）
               → 制度管理（明确"禁止"和"必须"）
               → 个体防护（最后防线）

[物] 不安全状态 → 工程技术（消除/预防/减弱/隔离/连锁/警告）
               → 制度管理（检查、检测、维保）

[环] 不良环境   → 工程技术（改善或消除环境条件）
               → 制度管理（监测预警、暂停作业条件）

[管] 管理缺陷   → 制度管理（完善制度流程）
               → 培训教育（让制度落地）
```

### 6.3 案例实操：管沟机械开挖（RP-01）的管控措施

以"机械分层开挖"步骤中"挖掘机距沟边过近"这个危险源为例：

```
危险源：挖掘机距沟边过近
潜在后果：沟壁坍塌 → 机械倾覆 → 人员伤亡

制定的措施：
① 工程技术[隔离] → 在挖掘机回转半径外设置警戒线或硬质围挡
② 制度管理      → 规定安全距离≥1.5m，设专职监护人员指挥
③ 培训教育      → 对司机进行安全操作规程培训
④ 应急处置      → 编制坍塌应急预案，配备应急支护材料
```

注意顺序：**先工程技术，再制度管理，再培训教育，最后才是应急处置。**

### 6.4 措施有效性评审

每条措施确定后，都要问三个问题：

1. **可行吗？**——在现有技术、资源条件下能落实吗？
2. **有效吗？**——能真正把风险降到可接受水平吗？
3. **会引入新风险吗？**——这个措施本身有没有副作用？

通过评审的措施才能正式纳入管控清单。

## 第7章 第五步：编制风险管控清单（出成果）

### 7.1 清单是什么？

经过前面四步（确定风险点→辨识危险源→评价风险→制定措施），我们把所有成果汇总到两张标准表格中，就是**风险分级管控清单**。

### 7.2 两张清单格式

**作业活动风险分级管控清单**（针对作业活动类风险点）：

| 风险点 | 工作步骤 | 潜在事故 | 危险有害因素 | 风险等级 | 管控措施 | 应急措施 | 管控层级 | 责任人 |
|--------|---------|---------|-------------|---------|---------|---------|---------|-------|
| 管沟机械开挖 | 施工准备 | 爆炸、触电 | 管线图纸不符、未办动土审批 | Ⅱ级/橙 | 管线保护交底+探测仪确认+动土审批 | 联络管线单位 | 项目部 | 项目负责人 |
| | 分层开挖 | 坍塌、机械伤害 | 挖掘机距沟边过近、未按设计放坡 | Ⅱ级/橙 | 安全距离≥1.5m+警戒区+专职监护 | 坍塌应急预案 | 项目部 | 项目负责人 |
| | 人工清底 | 坍塌、中毒窒息 | 沟壁悬空土块、有害气体积聚 | Ⅱ级/橙 | 敲帮问顶+气体检测+通风 | 有限空间应急预案 | 项目部 | 项目负责人 |

**设备设施风险分级管控清单**（针对设备设施类风险点）：

| 风险点 | 检查内容 | 标准要求 | 偏差事故 | 风险等级 | 管控措施 | 管控层级 |
|--------|---------|---------|---------|---------|---------|---------|
| 挖掘机 | 液压系统 | 管路无渗漏、油位正常 | 物体打击 | Ⅲ级/黄 | 每日班前检查、定期换油 | 班组级 |
| 挖掘机 | 安全防护装置 | 倒车影像有效、警灯正常 | 车辆伤害 | Ⅱ级/橙 | 每日检查、损坏立即维修 | 项目部 |
| 挖掘机 | 管沟边作业适配性 | 机身距沟边≥1.5m | 坍塌 | Ⅱ级/橙 | 划线管理、加垫路基箱 | 项目部 |

### 7.3 重大风险单独汇总

Ⅰ级重大风险要单独列出，由企业级管控：

| 风险点 | 事故类型 | 主要控制措施 | 管控层级 |
|--------|---------|-------------|---------|
| 边坡支护安拆（RP-02） | 坍塌、物体打击 | 专项方案+专家论证+每日检查 | 企业级 |
| 管道吊装下管（RP-03） | 起重伤害、坍塌 | 吊装方案+入场检测+全程旁站 | 企业级 |

### 7.4 成果清单总览

至此，我们已经完成了从风险点到管控清单的全过程：

```
资料准备 → 风险点确定（14个）
  → 危险源辨识（JHA+SCL）
    → 风险评价（LEC/LS）
      → 控制措施确定（五大措施）
        → 风险分级管控清单（最终产出）
```

## 第8章 第六步：编制隐患排查清单（控闭环）

### 8.1 核心逻辑：管控措施 = 排查内容

这是整个双重预防机制**最关键的一句话**。

你在风险管控清单中写下的每一条措施，都是隐患排查清单中需要检查的内容。换句话说：

> **风险管控清单说的是"该做什么"，隐患排查清单说的是"做了没有"。**

```
风险管控清单                        隐患排查清单
─────────────────                  ────────────────

风险点名称  ── 映射 ──→  排查单元（查哪个区域/活动）
危险源     ── 映射 ──→  排查项目（查什么方面）
管控措施   ── 映射 ──→  排查内容（查什么标准）
  工程技术措施                       检查工程措施是否落实
  制度管理措施                       检查管理措施是否落实
  培训教育措施                       检查培训是否完成
  个体防护措施                       检查防护用品是否佩戴
  应急处置措施                       检查应急物资是否到位
```

### 8.2 从管控措施到排查内容：一个实例

以管沟机械开挖（RP-01）为例，把管控措施转化为排查内容：

| 管控措施（来自风险管控清单） | 排查内容（用于隐患排查清单） |
|---------------------------|---------------------------|
| 工程措施：按方案放坡 | 检查是否按方案要求的坡度放坡，坡面是否平整 |
| 管理措施：挖掘机距沟边≥1.5m | 检查沟边安全距离是否满足，是否有划线管理 |
| 管理措施：设专职监护人员 | 检查监护人员在岗情况、指挥信号是否清晰 |
| 培训教育措施：作业前安全交底 | 检查交底记录，询问作业人员是否知晓风险 |
| 个体防护措施：佩戴安全帽 | 检查现场人员安全帽佩戴是否规范 |
| 应急处置措施：编制坍塌应急预案 | 检查应急预案是否制定、应急物资是否配备 |

### 8.3 隐患排查清单的分类

```
                  隐患排查清单
                      │
          ┌───────────┴───────────┐
          │                       │
   基础管理类排查清单         生产现场类排查清单
   （制度、培训、资质等）     （作业活动+设备设施）
                                   │
                          ┌────────┴────────┐
                          │                 │
                    作业活动类            设备设施类
                    排查清单             排查清单
```

**基础管理类**：检查企业和项目的安全管理体系是否健全（资质、机构、制度、培训、投入、应急等）。

**生产现场类**：检查现场的风险管控措施是否落实——这正是从风险管控清单转化过来的部分。

### 8.4 隐患排查频次

不同层级的排查，频次不同：

| 排查类型 | 组织者 | 频次 | 重点 |
|---------|-------|------|------|
| 日常排查（班中巡检） | 班组长/作业人员 | 每班 | 全覆盖 |
| 专项排查 | 项目部 | 按需 | 特定风险 |
| 综合性排查 | 项目部 | 每周 | 全面 |
| 综合性排查 | 企业 | 每月/每季 | 重大风险 |
| 复工前排查 | 项目部 | 停工后复工前 | 全面 |

### 8.5 隐患治理闭环

排查出隐患后，必须完成闭环管理：

```
发现隐患 → 登记台账 → 制定整改方案 → 实施整改
  → 验收确认 → 销号归档
```

**一般隐患**：立即整改或限期整改，整改后验收确认。

**重大隐患**：必须：
1. 立即暂停相关作业
2. 制定专项整改方案
3. 落实整改责任人、资金、时限
4. 整改完成后组织验收
5. 验收合格方可恢复作业

## 第9章 第七步：持续改进（活下去）

### 9.1 两张清单不是一成不变的

以下情形发生时，必须及时更新：

| 触发条件 | 做什么 |
|---------|-------|
| 进入新施工阶段 | 新增该阶段的风险点和管控措施 |
| 工艺、设备、材料变化 | 重新辨识和评价新增风险 |
| 周边环境变化 | 评估新增环境风险 |
| 发生安全事故 | 举一反三，补充管控措施 |
| 法规标准更新 | 调整风险等级判定标准 |
| 隐患排查发现新问题 | 分析原因，补充管控措施 |

### 9.2 日常运行机制

| 层级 | 频次 | 工作内容 |
|------|------|---------|
| 企业主要负责人 | 每半年 | 综合检查，分析重大风险管控效果 |
| 企业分管负责人 | 每季度 | 专业检查，补充完善清单 |
| 项目部 | 每日/每周 | 动态跟踪措施落实和隐患治理 |
| 班组 | 每班前 | 对作业环境、设备进行安全确认 |

### 9.3 信息化管理

利用信息系统实现：
- **风险管控**：清单电子化，随时查阅更新
- **隐患排查**：任务自动派发，整改跟踪，台账自动生成
- **统计分析**：趋势分析，预警提醒，辅助决策

## 结语：一个案例的完整闭环

让我们回顾一下，通过这个管沟开挖的案例，我们走完了怎样的旅程：

```
┌─────────────────────────────────────────────────────────────┐
│                    双重预防机制完整闭环                        │
│                                                             │
│  管沟开挖（一个工程）                                          │
│     ↓                                                        │
│  14个风险点（划范围）—— 找到"哪里可能出事"                      │
│     ↓                                                        │
│  危险源辨识（找根源）—— 搞清楚"什么原因导致出事"                  │
│     ↓                                                        │
│  风险评价（定级别）—— 判断"哪个最要命"                          │
│     ↓                                                        │
│  管控措施（定方案）—— 确定"该做什么来预防"                       │
│     ↓                                                        │
│  风险管控清单（出成果）—— 回答"风险在哪、怎么管"                  │
│     ↓                                                        │
│  隐患排查清单（控闭环）—— 检查"措施落实了没有"                   │
│     ↓                                                        │
│  排查 → 整改 → 验收（抓落实）—— 发现隐患就治理                    │
│     ↓                                                        │
│  持续改进（活下去）—— 动态更新，永不停止                        │
└─────────────────────────────────────────────────────────────┘
```

说一千道一万，三条核心原则：

1. **风险点要全**——覆盖所有作业、设备、场所，不留盲区
2. **管控措施要实**——每条措施具体、可操作、可检查
3. **排查内容要准**——直接对照管控措施进行验证，闭环管理

两张清单，一道防线是"预防"，二道防线是"验证"，两条腿走路，缺一不可。
